顺便感叹一下 Docusaurus 做网页确实方便，只要是框架之内的东西唰唰两下就改好了，“开箱即用”作为产品来讲是很有吸引力的。
但用来写博客其实也有不爽的地方，它的 markdown 又和 github 这些有区别。居中之类的一些常见格式调整要求用内置的 Infima 类，而且写岔了是要报错的……
